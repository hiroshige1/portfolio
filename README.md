# 廣重 歩 (Ayumu Hiroshige) - 学生エンジニア

はじめまして！プロフィールをご覧いただきありがとうございます。
東京科学大学（旧 東京工業大学）理学院数学系に在籍している**廣重 歩**です。

**純粋数学**（特に代数トポロジーや幾何学）に強い関心を持ちつつ、現在はプロフェッショナルとしてWebおよびネイティブアプリ開発に従事しています。学業と並行して、データ分析や機械学習を用いた業務の経験を積むことを目標としています。
現在は**幾何学的機械学習**に深い関心があり、修士課程ではこの分野の研究を進める予定です。
